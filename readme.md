# Netharpy websearch toolkit
Учебный проект для netology.ru

## HarpyCrawler
Программа обходчик сайтов

## HarpyServer
Вебсервер для работы с сайтом

## Среда разработки
Visual Studio 2022 Community Edition
Компилятор MSVC v143
Стандарт std::c++20

## Использованные библиотеки

- OpenSSL 3.2.1
- Wt 4.10.3
- Boost 1.82.0
- cURL 8.6.0
